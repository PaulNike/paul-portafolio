Learning Experience – Plataforma Educativa Rimac Seguros

Cliente: Rimac Seguros · Año: 2024 · Rol: Full-Stack Developer – Plataforma Educativa Empresarial

Contexto

Rimac Seguros buscaba reforzar su estrategia de formación interna construyendo una plataforma educativa robusta, escalable y personalizada para sus miles de colaboradores. La plataforma debía incluir cursos, instructores, temporadas, grupos, horarios, evaluaciones y una experiencia integral de aprendizaje.

Además, era indispensable integrar el sistema con autenticación corporativa del Directorio Activo (Azure AD), permitir acceso con Google y administrar grandes volúmenes de archivos en AWS S3.

Solución Implementada

Se desarrolló un CMS educativo completamente a medida utilizando Laravel, con arquitectura escalable distribuida en múltiples servidores, un panel administrativo avanzado y un front office dinámico optimizado para cursos presenciales y virtuales.

Front Office – Plataforma de aprendizaje

  • Interfaz intuitiva para colaboradores.
  • Acceso a cursos, información de instructores y horarios.
  • Inscripción a temporadas y grupos formativos.
  • Perfil del alumno con historial y progreso.
  • Contenido multimedia alojado en AWS S3.

Back Office – Panel administrativo

  • Gestión de banners, cursos y categorías.
  • Gestión completa de instructores y colaboradores.
  • Creación de temarios estructurados.
  • Administración de temporadas y sus grupos.
  • Control de horarios, sesiones y enrolamientos.
  • Gestión de inscritos por curso o temporada.
  • Generación de reportes académicos y administrativos.
  • Gestión avanzada de usuarios y roles.

Tecnologías Utilizadas

  • PHP – Framework Laravel
  • MySQL como base de datos estructurada
  • Bootstrap + jQuery para interfaces dinámicas
  • Autenticación corporativa con Azure AD (Directorio Activo)
  • Autenticación Google OAuth
  • AWS S3 para almacenamiento de contenido multimedia

Despliegue e Infraestructura

  • VPS Linux (Ubuntu) configurado para alto rendimiento.
  • Balanceador de carga distribuyendo tráfico entre 3 servidores de aplicación.
  • 1 servidor dedicado para base de datos MySQL.
  • Optimización de caching y respuesta de APIs.
  • Arquitectura escalable preparada para miles de usuarios concurrentes.

Resultados Clave

  • Una plataforma educativa escalable para miles de colaboradores.
  • Acceso simplificado gracias a SSO con Azure AD y Google.
  • Mayor eficiencia en la gestión de cursos y temporadas.
  • Aumento en la participación y seguimiento formativo.
  • Back office robusto que reduce carga operativa del área de capacitación.
