Fegasa – Tienda Virtual B2C

Cliente: Fegasa Perú · Año: 2023 · Rol: Diseño UX/UI – Full-Stack Developer

Contexto

Fegasa, una empresa peruana especializada en productos de línea industrial y de consumo, necesitaba transformar su antiguo catálogo web en una tienda virtual B2C moderna, rápida y enfocada en conversión. El objetivo principal era incrementar ventas directas, mejorar la experiencia del usuario e integrar una pasarela de pago robusta que permita operar de manera fluida desde cualquier dispositivo.

La plataforma debía ser administrable, escalable y lo suficientemente flexible para manipular miles de productos, categorías y listas de precio, sin afectar el rendimiento del sitio.

Solución Implementada

Se desarrolló una tienda virtual completa a medida, utilizando CodeIgniter como framework principal y enfocada en ofrecer una experiencia de compra clara, rápida y confiable. El diseño fue completamente renovado bajo un enfoque mobile-first.

Características del Front Office

  • Diseño y maquetación responsive orientado a conversión.
  • Buscador interno con filtros dinámicos por categoría, marca y precio.
  • Página de producto optimizada para SEO y velocidad.
  • Carrito de compras persistente.
  • Proceso de checkout rápido con validaciones en tiempo real.
  • Integración con pasarela de pago Mercado Pago.

Desarrollo del Back Office

  • Gestión de banners y contenido promocional.
  • Gestión completa de categorías y subcategorías.
  • Gestión de productos con variantes, precios y stocks.
  • Administración de marcas y líneas de productos.
  • Gestión de clientes y suscripciones.
  • Control de mensajes y solicitudes del formulario de contacto.
  • Configuración avanzada de fletes y costos de envío.
  • Gestión de listas de precio (minorista / mayorista / ofertas).
  • Panel de ventas con detalle de pedidos, pagos y seguimiento.

Tecnologías Utilizadas

  • PHP – Framework CodeIgniter
  • MySQL para la administración de productos y ventas
  • Bootstrap + jQuery (UI responsiva y dinámica)
  • Pasarela de pago Mercado Pago
  • Integración API REST para métodos de envío

Resultados Clave

  • Aumento significativo de ventas gracias a un proceso de compra más intuitivo.
  • Reducción del tiempo de carga mediante optimización de consultas y assets.
  • Control total del catálogo mediante un CMS robusto.
  • Incremento del tráfico orgánico gracias a una mejor estructura SEO.
  • Checkout seguro y confiable mediante Mercado Pago.
