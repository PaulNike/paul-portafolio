Beneficios – TASA

Cliente: TASA · Año: 2024 · Rol: Desarrollador Full-Stack – CMS a medida

Descripción del Proyecto

“Beneficios – TASA” es una plataforma web interna para empleados de TASA que centraliza y presenta todos los beneficios laborales disponibles según el perfil del trabajador (operarios, administrativos, otros). La plataforma fue desarrollada como un **CMS a medida** para gestionar contenidos dinámicamente, permitiendo a la organización administrar páginas, categorías, beneficios, temas, artículos, usuarios y archivos desde un panel personalizado.

El objetivo principal fue digitalizar y facilitar el acceso a la información sobre beneficios, así como permitir una gestión simple y flexible por parte del equipo de TASA.

Solución Implementada

Se implementó una aplicación web responsiva con backend en Laravel (PHP), una arquitectura modular en la que cada componente de contenido del CMS puede ser creado, editado o eliminado desde un panel de administración seguro.

El desarrollo incluyó tanto la interfaz pública para usuarios finales (empleados de TASA) como el panel administrativo con roles y permisos.

Desarrollo y Alcance

  • Diseño responsivo y programación frontend adaptado a múltiples dispositivos.
  • Backend robusto con Laravel para gestión de contenidos (CMS).
  • Panel administrativo completo para administración de todos los módulos.
  • Módulos de gestión: páginas, categorías, beneficios, temas y artículos.
  • Gestión de usuarios con roles y permisos.
  • Gestión avanzada de archivos y recursos multimedia.
  • Autenticación integrada con MSAL (Azure AD) para acceso seguro.
  • Despliegue en VPS Linux con base de datos MySQL optimizada.

Tecnologías Utilizadas

  • PHP (Laravel Framework)
  • MySQL (Base de datos)
  • Bootstrap (Frontend responsivo)
  • MSAL / Azure AD (Autenticación de usuarios)
  • HTML5, CSS3, JavaScript
  • VPS Linux (Ubuntu), servidor en la nube

Resultados y Beneficios

  • Entrega de una plataforma interna accesible desde cualquier dispositivo.
  • Permite administración completa del contenido sin intervención técnica.
  • Facilitó el acceso a los beneficios laborales para cientos de trabajadores.
  • Integración segura con el sistema de autenticación corporativo (MSAL).
  • Arquitectura extensible y mantenible para evolución futura.
